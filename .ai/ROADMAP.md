# Roadmap do Projeto Orion

## Estado geral

Somente a Fase 0 está iniciada. As demais fases são planejamento e não devem ser tratadas como implementadas.

## Fase 0 - Fundação

Status: iniciada.

Objetivos:

- repositório;
- memória permanente;
- documentação;
- decisões;
- segurança.

Critério de saída:

- documentação inicial revisada;
- decisões arquiteturais registradas;
- regras de desenvolvimento registradas;
- aprovação para criar a estrutura do monorepo.

## Fase 1 - Orion Core

Status: planejada.

Escopo:

- autenticação;
- usuários;
- setores;
- empresas;
- permissões;
- dashboard;
- auditoria.
